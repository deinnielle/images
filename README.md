# Instagram clone
Built in React, PHP and SQLite

<img src="https://media.giphy.com/media/4JY0RtY2Qid1NoRoxW/giphy.gif" width="100%">

## Install
```
$ git clone https://github.com/deinnielle/images.git
$ cd images
$ npm install
$ cd src
$ php -S localhost:1111
$ cd images
$ npm start
```
