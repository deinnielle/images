# Instagram clone
Built in React, PHP and SQLite

<img src="https://media.giphy.com/media/4JY0RtY2Qid1NoRoxW/giphy.gif" width="100%">

## Install
1. $ git clone https://github.com/deinnielle/images.git
2. $ npm install
3. $ cd src
4. $ php -S localhost:1111
5. $ cd ..
6. $ npm start
